Glossaires des institutions

Explorer le glossaire

Le glossaire répertorie toutes les institutions apparaissant dans les sources consultées. Même si ce répertoire comprend près de 650 entrées, il n’est sans doute pas complet. Des données descriptives ont été réunies pour chacune des institutions, parmi lesquelles leur nom et, lorsque l’information est connue, leurs dates d’ouverture et de fermeture.

Les institutions ont été géoréférencées, afin qu’elles apparaissent sur une carte. Pour chaque institution, des citations originales tirées des sources ont été rassemblées, contenant des informations supplémentaires sur le but de l’établissement et les personnes qui y étaient internées.

Glossaire des actes normatifs

Explorer le glossaire

Au niveau fédéral comme au niveau cantonal, il existait un grand nombre d’actes normatifs (lois, ordonnances ou règlements) sur lesquels les autorités cantonales compétentes pouvaient se fonder pour ordonner un internement administratif.

Ces actes pouvaient être de droit public, de droit civil ou de droit pénal, même si une grande partie d’entre eux relevaient du droit public (administratif) cantonal. Ces actes normatifs se concentraient sur différents groupes, par exemple les personnes de mauvaises mœurs, les réfractaires au travail, les buveurs ou alcooliques, les jeunes, les faibles d’esprit ou personnes souffrant de troubles psychiques, les vagabonds ou encore les prostituées. On trouve ici une liste des principaux actes cantonaux et fédéraux, en Suisse, qui ont un lien avec la pratique de l’internement administratif.

Toutes les données brutes et la documentation de chaque visualisation sont disponibles sur le portail opendata.swiss sous ce lien : opendata.swiss/fr/dataset/anstaltstopografie.

Sur ce portail, il est possible d’extraire les données suivantes :

  • Les sources originales en version PDF
  • La liste complète de toutes les institutions recensées (repères géographiques)
  • La topographie des institutions mentionnée dans les différentes sources
  • Le résumé des données informatiques pour le développement d’applications informatiques
  • La base de données spécifique de chaque visualisation
